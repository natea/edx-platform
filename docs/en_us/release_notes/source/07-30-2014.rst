****************************************
edX Release Announcements Mailing List
****************************************

These edX Course Staff Release Notes are always available and are updated with
each release of the edX Platform on `edx.org`_ and `edX Edge`_.

You can also sign up for the `edX Release Announcements mailing list`_ to get
an email message when we have updated the edX Course Staff Release Notes.

***************************************
edX Studio
***************************************

* **A bug seems to have caused a problem in Internet Explorer - ? PR#4392 - check with Cale, Ned, DB, Christina**

* A new **Correct or Past Due** setting in Problem components allows students to see the answer to a problem only if the student has answered the problem correctly or the problem's due date has passed.



***************************************
edX Learning Management System
***************************************
   
* On the edX.org sign-up page, the **Public Username** field is now above the **Password** field. 

* The appearance of the user profile area in the left pane of the student dashboard has changed.

.. Changes are pretty slight (see screen shot in PR#4453: https://github.com/edx/edx-platform/pull/4453); may not want to include this in release notes

* The display name for an open response assessment now appears in the course ribbon. (ORA-51)
